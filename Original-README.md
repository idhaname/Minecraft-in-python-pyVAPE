## Play
To play Minecraft, please type in a UNIX-like terminal first:
```shell
# Github
git clone https://github.com/Minecraft-in-python/Minecraft

# Gitee
git clone https://gitee.com/Minecraft-in-python/Minecraft

cd Minecraft
./install.py

# Run
python3 -m minecraft
```
